  2. Software Engineer

    FT Labs

    August 2011 to July 2014 (3 years)

    I joined FT Labs while it was still the startup Assanka which was bought by the Financial Times and became the web technology R&D group responsible for the development of the multi-award winning, cross-platform HTML5 and JavaScript FT web app.

    Initially I worked on the FT web app code base focussing on JavaScript, HTML5, CSS3 and the FT Labs front-end MVC framework Fruitmachine, with exposure to back-end PHP and Node.js. This included requirement definition and analysis, feature development, testing, bug reporting and fixing, daily standups and contributing to sprint planning.

The FT Windows app is a thin(ish) native Windows 8/8.1 wrapper I created using HTML and JavaScript around the FT web app.

I introduced functional testing using WebDriver, Jenkins and Java with a focus on continuous integration, best practice and striving for the main branch to never be broken. I architected a test suite implemented by myself and a team of 1-3 testers and encourage developers to take responsibility for creating further tests. I experimented with automated visual regression testing using the BBCs Wraith. During this time I also drove the development of the FT Firefox OS app and throughout continued working on the FT web app code base.
